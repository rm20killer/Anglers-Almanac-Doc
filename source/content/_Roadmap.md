> [!important] 
> This is a rough plan


# Phase 1: The Foundation (Pre-Launch) 
**Focus**: Asset Creation & Documentation

# Phase 2: Hytale Launch (Alpha)
**Focus:** Technical Stability
- **Barebones Release:** Launch the mod as a "Developer Tool" for server hosts and other modders.
	- Allow servers and modders to add their own loot/rods using the JSON format.
- **Loot Engine:** Finalize the Hierarchical Lookup system (Global > Zone > Biome).
- **Basic Almanac:** Implement the functional UI for the book (Name, Description, and Habitat tracking).

# Phase 3: The World overhaul (Beta)
**Focus:** Content Expansion & Gameplay Variety
- **Regional Saturation:** Populate every Zone and specific Region with loot
- **The Gear Overhaul:** Introduce the upgradeable Rods, Hooks, Reels, and Lines.
- **Minigames Overhaul**: Create more minigames that can be turned off and on

# Phase 4: The Deep Depths (Full Release)
**Focus:** End-Game Content & Polish
- **Boss Encounters:** Introduce the first **Boss Fish** - massive, multi-stage battles.
