## New feature
- Ability to spawn entities instead of items when fishing
- New rod (non craft able yet)
- New fish
- Fishing power feature
	- Higher fishing power makes higher rarity loot easy to catch while decreasing the chance for junk
- Integration with MMO skill tree 
	- Level and fish luck stat increase fishing power 
- New config settings
	- You can now turn off zone checks
	- You can now turn off enviroment checks
- Added German to languages *(translated by @drpepperaholic12)*
- Added in animation to fish cast and reel in
- API Improvements
	- Added in providers for loot and player stats

- Made it so bobber despawn if player is too far away.
- Fixed translation issue with bait
