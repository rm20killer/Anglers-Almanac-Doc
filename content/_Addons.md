# List of mods that have support for Angler’s Almanac

>[!info]
>**Developers**: if you added support for the mod please contact `RM20` on discord to get it added to this page.

# TaleQuests
**Type:** Questing Plugin

***TaleQuests** is a powerful, configuration first quest plugin built for **Hytale** servers.*

Features a dedicated `CATCH_FISH` quest type and fish tracking.

[Download on BuiltByBit](https://builtbybit.com/resources/talequests.90948/)

# Mermaids
**Type:** Gameplay Mechanics

***Mermaids** allows the player to transform into a Mermaid when entering the water!*

Adds the ability to fish up rare Mermaid Scales with a loot table expansion.
[View on CurseForge](https://www.curseforge.com/hytale/mods/mermaids)

# QuestLines Core

**Type:** Questing Plugin

***QuestLines** brings NPCs to life through a fully data-driven dialogue and quest system.*

Adds the ability to check when fish are caught  
[View on CurseForge](https://www.curseforge.com/hytale/mods/questlines)
# MMO Skill Tree
**Type:** RPG Progression

***MMO Skill Tree** adds an RPG-style progression system where every action you take helps you grow stronger.*

Almanac fishing actions grant specialized XP rewards scale dynamically based on your performance in the fishing minigame.
[View on CurseForge](https://www.curseforge.com/hytale/mods/mmo-skill-tree)
